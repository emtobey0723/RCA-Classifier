    How to Use This Tool

    This tool uses AI to classify Detractor interactions against the Vendor RCA Rubric, assigning a Primary, Secondary, and Tertiary driver to each case. Before running a classification, fill in all required case details and paste the full transcript for the interaction being reviewed. Follow the channel-specific guidance below to ensure the right content is submitted.

    Privacy & Data Notice

    PII and PHI are not stored anywhere when this tool is used. Transcripts are sent to Anthropic's API for real-time analysis only and are not retained, logged, or used for training after the API call completes. No transcript data is saved to any server or database. Session data (the classification log) exists only in your browser tab and is permanently deleted when the tab is closed.

    Phone
    Source
    CXOne

    Copy and paste the call transcript directly from CXOne into the transcript field. Ensure the full transcript is included from the start of the interaction through to the end of the call.

    Chat
    Source
    SFDC

    Copy and paste the chat transcript from Salesforce (SFDC) into the transcript field. Include the full conversation from the customer's opening message through the agent's closing.

    Email
    Source
    SFDC

    Paste each email response in chronological order, oldest to newest. Expand the oldest email in the thread first and paste everything in — this ensures any previous communications are captured and the AI has full context for an accurate classification.

    Required Fields

    All fields must be completed before a classification can be run:

    ✓ Agent Name
    ✓ Team Lead
    ✓ LOB
    ✓ Channel
    ✓ Case Number
    ✓ Date of Interaction
    ✓ Date of Survey Return
